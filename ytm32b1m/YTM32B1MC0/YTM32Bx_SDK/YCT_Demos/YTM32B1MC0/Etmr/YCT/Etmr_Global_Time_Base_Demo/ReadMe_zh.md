# Etmr_Global_Time_Base_Demo

---

## 硬件要求

- **EVB**: YTM32B1MC0-EVB-Q64

---

## 软件要求

- **SDK版本**: YTM32B1MC0_1_3_1 或更高版本
- **配置工具版本**: YCT v2.7.4 或更高版本

---

## EVB硬件设置

以下表格是例程中使用到的外部引脚配置：

|  引脚        |    功能       |
|  ------     |    -----      |
|  PTC_2      |    eTMR0_CH2  |
|  PTA_11     |    eTMR1_CH5  |

以下表格是EVB硬件连接设置，左端和右端短接：

|  左端        |    右端       |     描述        |
|  ------      |    -----     |     -----      |
|  J1-2        |    5VUSB     |     USB 5V供电  |
|  J2-2        |    5V        |     VDD 5V      |
|  J3-1        |    J3-2      |     单片机工作   |
|  J4-1        |    J4-2      |     VDDA 5V     |
|  J21-12      |     -        |     eTMR0_CH2   |
|  J20-9       |     -        |     eTMR1_CH5   |

---

## 功能描述

该例程展示了使用多个eTMR如何同时产生PWM信号。

1. 将逻辑分析仪或示波器接入J21-12，J20-9。
2. 逻辑分析仪或示波器中显示的两路PWM同步，频率为2kHz,占空比为50%。
