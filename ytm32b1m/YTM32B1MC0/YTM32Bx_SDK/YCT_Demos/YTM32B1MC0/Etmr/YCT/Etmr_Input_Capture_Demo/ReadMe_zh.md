# Etmr_Input_Capture_Demo

---

## 硬件要求

- **EVB**: YTM32B1MC0-EVB-Q64

---

## 软件要求

- **SDK版本**: YTM32B1MC0_1_3_1 或更高版本
- **配置工具版本**: YCT v2.7.4 或更高版本

---

## EVB硬件设置

以下表格是例程中使用到的外部引脚配置：

|  引脚        |    功能       |
|  ------     |    -----      |
|  PTC2       |    eTMR0_CH2  |
|  PTC3       |    eTMR0_CH3  |
|  PTB4       |    eTMR0_CH4  |
|  PTB5       |    eTMR0_CH5  |
|  PTE8       |    eTMR0_CH6  |
|  PTE9       |    eTMR0_CH7  |
|  PTB2       |    eTMR1_CH0  |
|  PTB3       |    eTMR1_CH1  |
|  PTC14      |    eTMR1_CH2  |
|  PTC15      |    eTMR1_CH3  |
|  PTC0       |    eTMR1_CH6  |
|  PTC1       |    eTMR1_CH7  |

其中eTMR0的通道2-7分别输出PWM信号，由eTMR1的通道0-3和通道6-7进行捕获。

---

## 功能描述

该例程展示了eTMR1的六路输入捕获功能。eTMR0输出PWM信号的频率为1kHz，占空比分别为99%，99%，99%，1%，1%和1%。

**注**：

1. 如果频率为2kHz，最小占空比依然为1%，那么最多只能设置4个通道；如果要设置更多的通道同时捕获，那么能捕到的最小占空比就会大于1%。具体根据实际情况来设置。
2. 该例程只考虑了1次溢出的情况，因此需要设置合适的预分频值，让计数器的值不超过1次溢出的情况。
