# Mpwm_Pwm_Demo

---

## 硬件要求

- **EVB**: YTM32B1MC0-EVB-Q64

---

## 软件要求

- **SDK版本**: YTM32B1MC0_1_3_1 或更高版本
- **配置工具版本**: YCT v2.6.3 或更高版本

---

## EVB硬件设置

以下表格是例程中使用到的外部引脚配置：

|  引脚        |    功能       |
|  ------      |    -----      |
|  PTE5        |    MPWM0_CH3   |

以下表格是EVB硬件连接设置，左端和右端短接：

|  左端        |    右端       |    描述          |
|  ------      |    -----     |    ----          |
|  J1-2        |    J1-3      |    USB 5V供电    |
|  J2-1        |    J2-2      |    VDD 5V        |
|  J3-1        |    J3-2      |    MCU工作       |

---

## 功能描述

该例程展示了MPWM2通道2和3的pwm输出功能。它们的频率为10KHz，初始占空比设置为25%和50%。然后通道3的占空比每隔10ms就会更新一次。
